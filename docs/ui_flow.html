4. Tổng Kết Luồng Điều Hướng
Từ màn hình → Đến màn hình Hành động
🚀 Splash → 🔐 Login / 🏠 Home Auto check auth
🔐 Login → 🏠 Home Dashboard Đăng nhập thành công
🔐 Login → 📝 Register Tap "Đăng ký"
🏠 Home → 📊 Kanban / 📂 Projects / 👥 Team / 📅 Calendar / 👤 Profile Bottom Tab Bar
📊 Kanban → 📝 Task Detail Tap on task card
📊 Kanban → ➕ Create Task Tap "+" button
📂 Project List → 📄 Project Detail Tap on project
📄 Project Detail → 📝 Task Detail / ➕ Create Task Tap task / Tap "+"
👥 Team → ✉️ Invite Member Tap "Invite"
🏠 Home → 🔔 Notifications Tap bell icon
